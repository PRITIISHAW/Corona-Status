# Corona-Status
1. Download the folder named corona-status, open the terminal and add a path to it.

2. Install all the necessary Packages listed below:
a) Tkinter
  b) Pillow
  c) Sqlite3
  d) Xlsxwriter
  e) Beautiful Soup
  f) Plyer
  g) Requests
  
3. Run "python main.py", to run the file.
